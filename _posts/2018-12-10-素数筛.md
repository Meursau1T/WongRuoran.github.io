```C
bool prime[10000001];
void pri(int end){
  memset(prime,1,sizeof(prime));  // 全为素数
  prime[0]=prime[1]=0;            // 排除特殊
  for(int i=2;i*i<=end;i++){
    if(prime[i]){
      for(int j=i*i;j<=end;j=i+j)
        prime[j]=0;
    }
  }
}
```
原理是素数的所有倍数都是合数。这种方法并不是最优解，比如2和3,它们有共同的倍数6,6会被重复操作，排除这种重复操作后的素数筛叫做线性素数筛。
